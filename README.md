# Intro 

This is a boilerplate including:
- angular 1.5.x 
- webpack 1.12.x
- babel 
- karma
- jasmine
- chai
- mocha 
- express 4
- basic node server
- supertest 


## webpack 

- uses index.template.html to dynamically create an appropriate index page in the build folder
- already comes with appropriate loaders for sass and Bootstrap 4 import 

## installation 

```bash
npm i 
npm start 
```
